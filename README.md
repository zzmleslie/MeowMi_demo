## ✨ 项目简介

本项目是一款以 **南京大学鼓楼校区** 真实校园为蓝本的 **2D 探索收集游戏**，由 **喵喵喵喵** 团队开发。

灵感来源于「南雍猫札」微信小程序——那个由同学们自发维护的校园猫咪记录平台。我们想在此基础上更进一步：让玩家化身为一只小猫 🐱，漫步在水彩风格的校园地图中，邂逅散落在校园各处的真实猫咪。每只猫咪都有完整档案——姓名、性格、亲属关系、绝育状态、收养情况，用技术为流浪猫留下温暖的数字痕迹。

> 🎯 本项目同时也是编程大赛参赛作品，融合了 **游戏开发 + 数据爬取 + 图像处理 + AI 智能体扩展** 等多维技术。

## 💡 设计理念

### 🎯 项目初心

南京大学鼓楼校区栖息着数十只流浪猫，牠们早已成为校园文化的一部分。同学们自发成立了猫咪关爱组织，开发了「南雍猫札」微信小程序，记录每只猫咪的信息——**这也是本项目最重要的灵感来源。**

然而这些数据大多停留在数据库里，缺少一个让人沉浸其中、真正"遇见"猫咪的入口。于是我们想：如果能把**整个鼓楼校园变成一张可探索的地图**，让玩家化身为一只小猫，漫步其间，邂逅每一只真实的校园猫咪——**让数据变得有温度，让记录变成体验。**

### 🏫 校园 × 游戏：虚实融合的探索

```
现实世界                        游戏世界
   │                              │
   ├── 📸 真实校园照片 ──────▶ 🗺️ 地图纹理数据
   ├── 🐱 真实猫咪信息 ──────▶ 🎮 猫咪 NPC + 图鉴
   ├── 📍 真实出没地点 ──────▶ 🎯 游戏内猫咪坐标
   └── 👥 真实社区活动 ──────▶ 💬 游戏内置社区
```

我们不满足于做一个"校园导览"，而是希望**用游戏的语言，讲好猫猫的故事**。

### 🎨 艺术方向：纯正 Undertale 像素RPG风格

本作在美术风格上全面致敬《Undertale》经典像素RPG美学：
- 🎮 **纯像素角色**：猫咪与玩家采用精致的黑白像素艺术，粗黑轮廓 + 纯色填充——复古、可爱、富有表现力
- 🖤 **黑白主调场景**：校园地图使用纯黑背景 + 暗绿像素草地 + 粗黑边框建筑，完整还原 Undertale 大地图氛围
- ♥ **红色灵魂点缀**：白色文字、红色爱心标记、白色像素对话框——经典的 Undertale 视觉符号贯穿全作

为什么选择这个风格？
- **Undertale 的情感力量**：极简的像素美学反而能承载最丰富的情感表达，和"记录猫咪"的温暖主题完美契合
- **像素 ≠ 粗糙**：精心设计 3px 粗边框 + 像素门窗 + 高光线，每一栋建筑都是手工像素艺术品
- **统一 ≠ 单调**：通过黑白主调 + 灵魂红点缀 + 土黄路径 + 深绿树冠，构建层次分明的像素世界
- **复古 RPG 沉浸感**：禁用抗锯齿、像素化纹理、扫描线叠加、等宽像素字体——打造原汁原味的 1990s RPG 体验

### 🐱 玩家身份设计

> **你是一只小猫。** 不是人类，不是游客，就是一只猫。

这个设计有几个巧思：
- 🐾 **平等视角**：你和其他猫咪是同类，相遇更自然
- 🎭 **身份代入**：降低"第三人称旁观"的距离感
- 💡 **交互隐喻**：小猫靠近小猫 → "打招呼"，比"人类走过去摸猫"更温柔

### 🧩 核心玩法循环

```
  探索校园 ──→ 发现猫咪 ──→ 查看档案 ──→ 收集成就
     ↑                                      │
     └──────────── 继续寻找下只猫 ←──────────┘
```

借鉴了"宝可梦式探索 + 图鉴收集"的经典玩法，并融入 Undertale 式的叙事风格，但做了减法：
- ❌ 不抓猫、不战斗——我们不是驯服者，是记录者
- ❌ 不强制教程——走就完事，发现是自然而然的事
- ✅ 靠近自动提示——不用刻意寻找
- ✅ 点击查看详情——想了解就了解，不想就继续逛

### � 剧情探索：每只猫都有自己的故事

> 受 **《Undertale》** 叙事风格的启发，我们不满足于只做"猫咪图鉴收集器"——**每一只猫咪都应该拥有一条属于自己的故事线。**

| 故事要素 | 说明 |
|----------|------|
| 🐱 **初见场景** | 每只猫有独特的"出场方式"——在北大楼台阶晒太阳、在食堂后门偷吃、在礼堂梧桐树下打盹…… |
| 📝 **背景故事** | 从流浪到被社团关爱、绝育、找领养——基于真实经历改编的温暖小故事 |
| 💬 **性格对话** | 根据猫咪真实性格设计对话树——傲娇猫嘴硬心软、胆小猫需要慢慢靠近、黏人精主动蹭过来 |
| 🔗 **关系网络** | 猫咪之间的亲属/敌对/好友关系，发现关联会触发隐藏剧情 |
| 🏆 **剧情收集** | 解锁全部故事线 → 获得完整图鉴与特殊成就 |

### 🎵 音乐 & 美术：打造沉浸式猫咪世界

视听是叙事的重要载体，我们计划从以下几个层面营造氛围：

| 层面 | 计划 | 状态 |
|------|------|:----:|
| 🎨 **像素角色** | 猫咪 & NPC 采用 Undertale 式像素风，萌系可爱、富有表现力 | 📋 |
| 🖼️ **水彩场景** | 校园地图使用程序化水彩渲染，温柔文艺 | 🔨 |
| 🎵 **背景音乐** | 原创/改编 Lo-fi 轻音乐，不同校园区域有不同变奏 | 📋 |
| 🔊 **环境音效** | 猫咪叫声、脚步声、风吹树叶、鸟鸣……增强临场感 | 📋 |
| 🌸 **季节主题** | 未来可考虑春樱、夏夜、秋叶、冬雪等季节视觉 | 💡 |

### �💬 社区：不止是游戏

猫咪的生存状况需要真实的人来关注。我们在游戏内置了社区模块：

| 板块 | 现实意义 |
|------|----------|
| 📷 **晒照** | 鼓励同学们拍猫、分享，形成UGC内容飞轮 |
| 🗣️ **讨论** | 交流猫咪动向、健康状态 |
| 💰 **捐款** | 为绝育/医疗筹集资金，透明公示 |
| 🙋 **志愿者** | 组织线下绝育/投喂活动 |

> 游戏的终点不是收集完数据，而是**让更多人参与到关爱猫咪的行动中**。

### 🤖 未来：AI 猫咪智能体

每只猫咪都有自己独特的性格（胆小、黏人、高冷…），我们预留了 AI 接口，未来可以：
- 🗣️ **猫咪对话**：用户可以向特定猫咪提问，AI 模拟猫咪的"语气"回答
- 🗺️ **智能导航**：AI 根据天气/时间推荐"今天去哪看猫"
- 📝 **自动更新档案**：社区帖子中提取信息，自动更新猫咪状态

### 🌟 一句话总结

> **用代码画一座校园，放几只猫，让路过的人停下来，看一眼。** ——这就是我们想做的事。

## 🚀 快速开始

### 环境要求

| 软件 | 版本 | 用途 |
|------|------|------|
| Unity Hub + Editor | 2022.3 LTS | 游戏开发 |
| Python | 3.9+ | 数据爬取 |
| Node.js | 16+ | 照片预处理（可选） |
| 浏览器 | Chrome / Edge | 测试 WebGL 版本 |

### 1. 安装 Unity

```text
1. 下载 Unity Hub: https://unity.com/download
2. Unity Hub → Installs → Install Editor
3. 选择 「Unity 2022.3 LTS」（⚠️ 不要选新版！）
4. 勾选 WebGL Build Support
5. 等待安装完成（约 3-5GB）
```

### 2. 创建 & 配置项目

```text
1. Unity Hub → New Project → 2D Core → 命名 MeowMeowMeowMeow
2. 将 unity-game/Assets/Scripts/ 全部复制到 Unity 项目的 Assets/ 中
3. 将 data/cats.json 和 data/map-data.json 放到 Assets/Resources/Data/
4. Package Manager 安装: com.unity.nuget.newtonsoft-json
5. 按照 SETUP_GUIDE.md 搭建场景
6. 点 ▶️ Play 运行！
```

> 📖 详细图文教程见：[unity-game/SETUP_GUIDE.md](unity-game/SETUP_GUIDE.md)

### 3. 爬取猫咪数据（可选）

```bash
cd scripts
pip install -r requirements.txt
python scraper.py --gui
```

### 4. 导出 WebGL

```text
File → Build Settings → WebGL → Switch Platform → Build
→ 输出文件夹部署到任意静态服务器
```

## 🤖 AI 智能体扩展规划

`GameManager.cs` 已预留两个 AI 接口，后续对接 LLM 即可：

```csharp
// 1. 猫咪 AI 助手 - 与猫咪"对话"
string reply = await GameManager.Instance.QueryCatAI("cat_001", "今天心情怎么样？");
// → "喵~今天太阳很好，我在北大楼晒了好久！"

// 2. AI 路线推荐
var route = await GameManager.Instance.GetRecommendedRoute();
// → ["北大楼→大黄", "大礼堂→雪球", "食堂→小橘"]
```
